# What Makes a Game Successful? 
### Analyzing Steam Game Data
This project explores which factors contribute to the success of video games on Steam, using data analysis and visualization.

---

## Project Overview

The video game industry is one of the largest entertainment sectors in the world â€” yet the factors that determine whether a game succeeds are often unclear.  
This project analyzes a dataset of Steam games to uncover patterns and relationships between:

- Price  
- Genre  
- Release date  
- Developer type (indie vs. AAA)  
- User reviews and ratings  

The goal is to identify which characteristics are most strongly associated with highly rated or popular games.

---

##  Research Questions
1. Which genres tend to have the highest ratings?
2. Does game price affect user satisfaction?
3. How have player preferences changed over time?
4. Are indie games rated differently from major publisher titles?

---

##  Data Sources
  
### Primary Dataset
- **Dataset:** [Steam Games Dataset on Kaggle](https://www.kaggle.com/datasets/artermiloff/steam-games-dataset)
#### The project will focus on data points such as:
  - Game title  
  - Developer and publisher  
  - Release date  
  - Price  
  - Review counts and user ratings

## Data Collection Plan

1. Download and inspect the dataset from Kaggle.  
2. Clean and preprocess the data (handle missing values, convert data types, etc.).  
3. If needed, combine additional open datasets for data enrichment.  
4. Prepare the final dataset for exploratory analysis and visualization.
